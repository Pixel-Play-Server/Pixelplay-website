Guía de Implementación Móvil

Paso 1: Agregar CSS de Optimizaciones

<!-- En el <head> de tu HTML -->
<link rel="stylesheet" href="assets/css/mobile-optimizations.css" />

Paso 2: Agregar JS de Optimizaciones

<!-- Antes del </body> cierre de etiqueta -->
<script src="assets/js/mobile-optimizations.js"></script>

Paso 3: Verificar Meta Viewport

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

Paso 4: Agregar div para Shell Nav

<div id="pp-shell-nav"></div>

Ejemplo Completo

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Mi Página - PixelPlay</title>
  
  <!-- Estilos -->
  <link rel="stylesheet" href="assets/css/mobile-optimizations.css" />
</head>
<body>
  <!-- Navegación dinámica -->
  <div id="pp-shell-nav"></div>
  
  <!-- Contenido principal -->
  <main>
    <h1>Mi Contenido</h1>
    <p>Lorem ipsum dolor sit amet...</p>
  </main>
  
  <!-- Footer dinámico -->
  <div id="pp-shell-footer"></div>
  
  <!-- Scripts -->
  <script src="assets/js/site-shell.js"></script>
  <script src="assets/js/mobile-optimizations.js"></script>
</body>
</html>

¿Cómo Probar?

  1. Abre el navegador (Chrome, Firefox, Safari)
  2. Presiona F12 para abrir DevTools
  3. Click en el icono de teléfono (Toggle Device Toolbar)
  4. Prueba diferentes dispositivos:
    • iPhone 12 (390x844)
    • Samsung Galaxy S20 (360x800)
    • iPad (768x1024)
  5. Verifica que:
    • El menú se abre y cierra
    • Los botones son grandes y clickeables
    • No hay scroll horizontal
    • El texto es legible
    • Las imágenes se cargan

¿Qué Sucede Automáticamente?

  • site-shell.js: Menú móvil, navegación, eventos táctiles
  • news.js: Feedback visual en artículos, optimización de búsqueda
  • post.js: Lazy loading, tablas responsivas, imágenes optimizadas
  • mobile-optimizations.js: Detección de dispositivo, rendimiento global
  • mobile-optimizations.css: Estilos responsivos globales

Solución de Problemas

El menú no aparece

  • Verifica que `id="pp-shell-nav"` exista en tu HTML
  • Abre la consola (F12 > Console) y busca errores
  • Asegúrate de que site-shell.js se carga después del HTML

El CSS no se aplica

  • Verifica que el path sea correcto: `assets/css/mobile-optimizations.css`
  • Abre DevTools > Network > y busca el archivo
  • Si no se carga, verifica los permisos del archivo

Las imágenes se ven pequeñas

  • Asegúrate de que el CSS `mobile-optimizations.css` se cargue
  • Verifica que las imágenes tengan `style="max-width: 100%"`
  • Usa el atributo `loading="lazy"` para mejor rendimiento
